Sum of Two Numbers

Difficulty: Easy

Description

Write a program that reads two integers and prints their sum.

Input Format

Two integers a and b separated by a space.

Output Format

A single integer, the sum of a and b.

Constraints

-10^9 ≤ a, b ≤ 10^9

Sample Test Cases

Input:

Output: 
10

1000000000 -1000000000
Output: 
0
